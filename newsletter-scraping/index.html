💡 Claude Tips

The Basics

  • Always inject today's date. This is the single biggest fix for outdated content. Every prompt, every time.
  • Paste the direct URL to the specific page (events page, careers page) — not the homepage. Claude reads exactly what you give it.
  • One content type per prompt. Don't ask Claude to find jobs AND events AND grants all at once. Focused tasks get better results.
  • Ask for table format. Tables are easy to review and paste straight into the Google Sheet.

The Chrome Extension

The Claude Chrome extension captures the current page and sends it to Claude. It works great for most sites, but can be inconsistent on:

  • Side-panel sites (like AERA's job board) — listings load in a popup sidebar when you click them. The extension only captures the main list view, not the panel content.
  • JavaScript-heavy pages — content that loads dynamically after the page opens.

Workaround order of operations:

  • ① Try the Chrome extension first
  • ② If it misses content — scroll through the full page to load everything, then try again
  • ③ If still incomplete — manually copy-paste the page text into Claude
  • ④ If the site consistently fails — try Firecrawl (see next section) or log it in the Problem Sites tab

Getting Better Results Over Time

  • When Claude returns something wrong, correct it in the same chat: "This posting is from 2024, it's expired — exclude anything without a clear current deadline." It adjusts immediately.
  • If a prompt works really well on a specific site, note it in your changelog so you remember to reuse it.
  • Don't try to scrape all 270+ orgs at once. Batch by tab — run foundations one session, think tanks another, job boards another.
🔥 Firecrawl (Optional Level-Up)

If the Chrome extension or copy-paste approach isn't working on a particular site, Firecrawl is your next option. It's not required — but it directly solves the most common failure mode.

What Is It?

Firecrawl (firecrawl.dev) fetches a webpage and converts it into clean text that AI can actually read — including sites that use JavaScript to load content dynamically. Free to use.

How to Use It (3 Steps)

1

Go to firecrawl.dev and paste the URL of the page you want to scrape.

2

Firecrawl returns the clean text content of that page.

3

Paste that content into Claude with your usual prompt:
"Today is [DATE]. Here is the content from [org]'s events page. Find all upcoming events..."

Start without it. Get the basic workflow running first. Pull in Firecrawl only for the sites that consistently give Claude trouble. You don't need it for everything — just the problem pages.

Want to Go Deeper?

Firecrawl also has an API that can automatically crawl entire websites. That's optional advanced territory — the free copy-paste version is all you need to get started. If you're curious about the API, let Kate know and you can explore it together.

Claude Code

You mentioned wanting to improve your Claude skills beyond just using claude.ai. Claude Code is the natural next step — and this newsletter project is actually a great sandbox for it.

What Is Claude Code?

Claude Code is a command-line tool made by Anthropic (same company as Claude). Instead of chatting in a browser, you work with Claude directly in your terminal. The big difference: Claude Code can read and write files on your computer, run code, and take multi-step actions on your behalf — not just answer questions.

Think of it as the difference between asking Claude to describe how to do something vs. having Claude actually do it while you watch.

Why It's Relevant for This Project

Right now the workflow is manual: you visit a page, use the Chrome extension or copy-paste, run a prompt, copy the output into the Google Sheet. Claude Code can change that. For example, you could tell it:

"Write a Python script that visits each of these 10 URLs, extracts all job postings, filters out anything older than 90 days, and saves the results to a CSV file."

Claude Code would write that script, run it, and hand you the CSV. No copy-paste, no tab-switching. That's the difference in practice.

It's also great for the problem sites — Claude Code with a browser tool can handle JavaScript-heavy pages that the Chrome extension can't.

How to Get Started

1

Make sure Node.js is installed

Check by opening Terminal and typing node --version. If nothing comes back, download it from nodejs.org (the LTS version).

2

Install Claude Code

In Terminal, run: npm install -g @anthropic-ai/claude-code

3

Launch it

Navigate to a folder in Terminal (e.g., cd Desktop), then type claude. You'll be in an interactive session where you can give it tasks.

4

Try a simple task first

Ask it to write a script that reads the Ed Policy Org List CSV and outputs just the foundation URLs. Small, concrete tasks are the best way to learn how it works.

The Learning Path

StageToolWhat You're Doing
Starting outClaude.ai ProjectsManual scraping with prompts + Chrome extension. Build the workflow, learn what works.
Level upClaude CodeAsk Claude to write scripts that automate the repetitive parts. Learn basic Python along the way.
AdvancedClaude Code + Firecrawl APIFully automated pipeline: scrape all sources, filter, output to spreadsheet. Minimal manual work.
You don't need coding experience to start with Claude Code. You can ask it to explain every line it writes, ask it to teach you what a function does, or ask it to rewrite something more simply. It's genuinely one of the best ways to pick up coding skills because you see working code immediately and can ask questions about it in context.
