3. Create an RSVP Form

Use Google Forms or Microsoft Forms for RSVPs. Check responses regularly to adjust food count, room size, and dietary planning.

Standard Fields

Adjust based on the event — skip dietary questions if no food is provided. Always include the listserv checkbox.

  • Name
  • Email address
  • Role (student / faculty / staff / community)
  • Department / college
  • Food allergies / dietary needs (if food is provided)
  • Checkbox: "Add me to the ERC listserv"
  • Checkbox: "Will you attend via Zoom?" (speaker/lecture events only)
How to Use RSVP Data
  • Monitor responses up to 1 week before the event
  • Use headcount to finalize catering order quantities
  • Flag anyone with dietary restrictions and note it on the catering order
  • If Zoom attendees are high, make sure room A/V is set up for hybrid

5. Food & Catering

Only complete this section if food is confirmed for the event.

Step 1: Submit a TLAC Purchase Request

Before ordering anything, you need purchase approval. Submit a TLAC Purchase Request with the following information. Andy can provide a step-by-step walkthrough.

  • Price estimate
  • Estimated headcount
  • Event timing (date + start/end time)
  • Event title and description
Step 2: Order Catering

After the purchase request is approved, order catering. The easiest and lowest-risk option is EZCater (delivery included).

EZCater Login

URL: ezcater.com/sign_in
Email: akwok@tamu.edu
Password: tamuERC01!
⚠ 2FA Required: Slack Andy before logging in so he can approve the authentication code from his phone or email.

Alternative: pick up food yourself — but only as a last resort, especially for larger events.

Food Logistics
  • Confirm delivery address includes: building name + room number + best entrance
  • Set food on a designated table with napkins, utensils, and a trash can nearby
  • Label dietary items where possible (vegetarian, gluten-free, etc.)
  • End of event: quick cleanup — bag trash and decide on leftover plan
6. Day-of Guide
Parking

Mention parking in the 24-hour reminder email if the event is on central campus or at Gilchrist.

  • Central campus events: Gene Stallings Garage — validation passes available
  • Gilchrist events: Fan Field Parking Lots
  • Parking passes: get from Andy or Kate, or request through Robin Burns via the ERC account
  • Bring passes to the event and be ready to direct attendees where to pick them up
AV & Zoom

Before the event, confirm the room has what you need:

  • Projector / screen
  • HDMI / USB-C adapter
  • Microphone (for larger rooms)
  • Podium
  • Tables for food + check-in

Bring (or confirm someone is bringing): laptop charger, adapter, clicker, tape, markers, extra pens. Usually handled by Kate.

Hybrid / Zoom Events
  • Use the ERC standing Zoom link — no new link needed
  • Test audio and video before the event starts
  • Assign someone (GA) to monitor the Zoom chat and relay questions
  • Confirm recording consent with the speaker before starting the recording
Setup & Roles

Arrive 30 minutes early. Assign roles before the event starts:

RoleResponsibilitiesUsually
Greeter / check-in Welcome attendees, direct them to food and seating GA
Food setup lead Receive catering delivery, set up table, label dietary items KB or GA
A/V helper Slides, mic, Zoom monitoring, recording AK or GA
Runner Last-minute tasks, extra supplies, anything unexpected GA
Cleanup
  • Bag all trash and remove from the room
  • Box or dispose of leftovers
  • Return chairs, tables, and AV equipment to original positions
  • Collect any ERC signage, name tags, or printed materials
  • Stop Zoom recording; save and rename the file if applicable
  • Thank the speaker personally before they leave
7. After the Event

Complete these within 1–3 days of the event.

  • Record basics: headcount, any issues, and what to do differently next time
  • Send a short thank-you email to the speaker (if applicable)
  • Post a recap to social media (photo + short description) if appropriate
  • Upload recording to the ERC website or shared drive (if recording was taken with consent)
  • Submit any reimbursement forms for speaker travel or catering expenses
  • Save the final blurb / URL / RSVP link for the next event of the same type
  • Mark all event tasks as Done in the Tasks page

Task Due Dates

Tasks use an offset relative to the event date:

  • E–30 = 30 days before the event
  • E+1 = 1 day after the event
  • Day of = the event date itself (offset 0)

Offsets are set in the templates sheet of the planning spreadsheet and can be adjusted there at any time.
